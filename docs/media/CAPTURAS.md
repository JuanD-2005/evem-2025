# Guía de Capturas para README

Este directorio contiene las imágenes usadas en el README.

## Archivos requeridos

- `home.png`
  - Vista general de la página principal (`index.html`).
  - Debe incluir hero y navegación institucional.

- `evem-form.png`
  - Formulario de inscripción EVEM (`pages/contact.html`).
  - Mostrar campos principales y estado visual del formulario.

- `dim-landing.png`
  - Landing oficial DIM (`pages/dim.html`).
  - Incluir bloque principal y estilo característico del evento.

- `admin-panel.png`
  - Panel administrativo DIM (`pages/admin-dim.html`).
  - Mostrar tabla/listado de inscritos y controles de pago.

## Recomendaciones

- Formato: PNG.
- Resolución sugerida: 1600x900 o 1920x1080.
- Mantener interfaz limpia (sin datos sensibles).
- Si hay datos personales en formularios, usar datos de prueba.
